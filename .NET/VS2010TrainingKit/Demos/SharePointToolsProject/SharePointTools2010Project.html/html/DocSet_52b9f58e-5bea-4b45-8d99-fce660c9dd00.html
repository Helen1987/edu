Opening Statement
 


In this demo, we will start using the Visual Studio 2010 SharePoint Developers Tools. This will give you a chance to become familiar with the standard project structure used by SharePoint Developers Tools. We will go through the steps of creating and testing a project that contains a Feature, and a Feature Receiver. Along the way, you will learn to configure the SharePoint project’s deployment options, as well as debug a SharePoint project using Visual Studio 2010 by single stepping through the code in your solution.

The following is a brief description of the topics that will be shown during this demo:

  1. Creating a SharePoint Project
  2. Adding a Feature Event Receiver
