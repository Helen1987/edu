Opening Statement
 


Today I would like to walk you through several demo applications built to show the new Barrier class of the System.Threading namespace. This very simple demo relies on the .NET Framework 4 release.

The following is a brief description of the things that will be shown during this demo:

  1. Exploring the Barrier class methods and properties
  2. Canceling the thread execution with the Cancellation Token
