In this demo you will see how to add a custom Form Region in Office 2010.

The Add-In adds a Custom Form Region to the New Task Windows in Outlook 2010. 

To run the demo, proceed as follows:
 	1- Run the Add-In, it will open Outlook 2010
	2- In the Home tab of the Ribbon, choose New Items \ Task
	3- At the bottom of the New Task windows you will find the Custom Form Region
