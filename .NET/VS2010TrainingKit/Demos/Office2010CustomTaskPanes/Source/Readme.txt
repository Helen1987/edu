In this demo you can see how to implement a custom task pane for Office 2010.

In the solution, you will find the TaskPane.xaml file which contains the definition of the custom task pane that will be rendered in Word 2010. 

In the task pane, you will see a list of text-snippets listed. Double-clicking on a snippet will replace the selected text with the chosen text-snippet.

Running the Solution
--------------------
To run the solution build it first, and then click the 'Debug' menu and select 'Start Without Debugging'.
After closing Word 2010, clean the solution to remove the custom task pane.
